# get-noa

Installer for the NOA CLI tool for [nanofleets.com](https://nanofleets.com)

## Installing

**Linux/macOS:**

```sh
curl -sSL https://raw.githubusercontent.com/nanofleets/get-noa/main/install.sh | bash
```

**Windows (PowerShell):**

```powershell
iwr -useb https://raw.githubusercontent.com/nanofleets/get-noa/main/install.ps1 | iex
```
